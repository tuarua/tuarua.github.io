public function vibrate(milliseconds:int = 0, pattern:Array = null, repeat:int = -1):void

Vibrate with a given pattern.

Pass in an array of ints that are the durations for which to turn on or off the vibrator in milliseconds. The first value indicates the number of milliseconds to wait before turning the vibrator on. The next value indicates the number of milliseconds for which to keep the vibrator on before turning it off. Subsequent values alternate between durations in milliseconds to turn the vibrator off or to turn the vibrator on.

To cause the pattern to repeat, pass the index into the pattern array at which to start the repeat, or -1 to disable repeating.

Parameters

milliseconds:int (default = 0) — the number of milliseconds to vibrate. Set to 0 when using a pattern. Ignored on iOS
 
pattern:Array (default = null) — an array of times for which to turn the vibrator on or off. Ignored on iOS
 
repeat:int (default = -1) — the index into pattern at which to repeat, or -1 if you don't want to repeat. Ignored on iOS
