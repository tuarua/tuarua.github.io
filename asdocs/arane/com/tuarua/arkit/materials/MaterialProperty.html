intensityproperty 
intensity:Number

Determines the receiver's intensity. This intensity is used to modulate the properties in several ways. It dims the diffuse, specular and emission properties, it varies the bumpiness of the normal property and the filter property is blended with white.

The default value is 1.0.

Specifies the filter type to use when rendering the the contents (specified in the `contents' property).

The magnification filter is used when to increase the size of image data.

mappingChannelproperty 
mappingChannel:int

Determines the receiver's mapping channel.

Geometries potentially have multiple sources of texture coordinates. Every source has a unique mapping channel index. The mapping channel allows to select which source of texture coordinates is used to map the content of the receiver.

The default value is 0.
