Parameters
keyName:String — The name of the custom attribute for use as a key in a SearchableItemAttributeSet. The key name must be a string that contains only ASCII characters and no punctuation other than the underscore (that is, “_”). The prefix kMD is reserved.
 
isSearchable:Boolean (default = true) — A Boolean value that indicates if the attribute can be specified as a search term.
 
isSearchableByDefault:Boolean (default = false) — A Boolean value that indicates if the attribute should be searchable by default.
 
isUnique:Boolean (default = false) — A Boolean value that indicates if duplicate values should be treated as the same value to save storage space.
 
isMultiValued:Boolean (default = false) — A Boolean value that indicates if the attribute is likely to have multiple values, such as arrays, associated with it.
