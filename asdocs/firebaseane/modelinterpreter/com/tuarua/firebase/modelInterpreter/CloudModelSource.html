public function CloudModelSource(name:String, enableModelUpdates:Boolean, initialConditions:ModelDownloadConditions, updateConditions:ModelDownloadConditions = null)

Creates an instance of CloudModelSource with the given name and the download conditions.

Parameters
name:String — The name of the model to download. Specify the name you assigned the model when you uploaded it to the Firebase console. Within the same Firebase app, all cloud models should have distinct names.
 
enableModelUpdates:Boolean — Indicates whether model updates are enabled.
 
initialConditions:ModelDownloadConditions — Initial downloading conditions for the model.
 
updateConditions:ModelDownloadConditions (default = null) — Subsequent update conditions for the model. If it is null and `enableModelUpdates` is true, the default download conditions are used.
