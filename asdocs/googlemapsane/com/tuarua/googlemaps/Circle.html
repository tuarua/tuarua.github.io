The width of the circle's outline in screen pixels. The width is constant and independent of the camera's zoom level. The default value is 10.

visibleproperty 
public var visible:Boolean = true

Indicates if the circle is visible or invisible, i.e., whether it is drawn on the map. An invisible circle is not drawn, but retains all of its other properties. The default is true, i.e., visible. Ignored on Apple Maps

zIndexproperty 
public var zIndex:uint = 0

The order in which this tile overlay is drawn with respect to other overlays (including GroundOverlays, TileOverlays, Polylines, and Polygons but not Markers). An overlay with a larger z-index is drawn over overlays with smaller z-indices. The order of overlays with the same z-index is arbitrary. The default zIndex is 0. Ignored on Apple Maps
